# ConvertorASCII

ConvertorASCII — это утилита для преобразования изображений в ASCII-арт. Программа конвертирует JPEG и PNG файлы в текстовое представление с использованием символов ASCII.

## Возможности

- **Поддержка форматов** - работа с JPEG и PNG изображениями
- **Настраиваемый размер** - регулировка ширины ASCII-арта
- **Коррекция пропорций** - настройка соотношения сторон для правильного отображения
- **Автоматическая обработка путей** - поддержка путей с пробелами и в кавычках

## Использование

### Запуск программы

После запуска доступны следующие команды:
Конвертировать [1]
Изменить настройки [2]
Закрыть [3]
Выбор команды:

### Конвертация изображений [1]

1. Введите путь к файлу изображения
2. Программа автоматически обработает файл и выведет ASCII-арт
3. Поддерживаются только файлы с расширениями `.jpg` и `.png`

### Настройки [2]

#### Максимальная ширина арта [1]
- Задает максимальную ширину ASCII-арта в символах
- Рекомендуемые значения: от 40 до 400
- Минимальное значение: 1, максимальное: 750

#### Соотношение сторон [2]
- Регулирует вытянутость арта для коррекции пропорций
- Рекомендуемое значение: 2.5
- Диапазон: от 0.5 до 5

## Технические особенности

- Используется алгоритм усреднения яркости пикселей
- Поддерживается 19 градаций яркости от темного к светлому
- Автоматическое масштабирование с сохранением пропорций
- Оптимизированная обработка больших изображений

## Важные примечания

- **Рекомендуемый формат** - для максимальной надежности рекомендуется использовать формат JPG
- **Кодировка путей** - пути не должны содержать русские символы во избежание проблем с кодировкой
- **Шрифты консоли** - для правильного отображения ASCII-арта используйте моноширинные шрифты
- **Качество изображений** - для лучшего результата выбирайте изображения с хорошим контрастом и четкостью
- **Производительность** - обработка больших изображений и высокого разрешения может занять время
- **Размер арта** - слишком маленькая ширина арта может привести к потере деталей изображения
